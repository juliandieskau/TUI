# ftxui-starter

CMakeList and main() method based on ftxui-starter:  
Minimal starter project using the [FTXUI library](https://github.com/ArthurSonzogni/ftxui)

# Build instructions:

~~~bash
mkdir build
cd build
cmake ..
make -j
cd ../target
./ftxui-starter
~~~
